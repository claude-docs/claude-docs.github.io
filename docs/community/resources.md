## Cheat Sheets

### CLI Commands

| Command | Action |
|---------|--------|
| `claude` | Start session |
| `claude -c` | Continue last |
| `claude -p "..."` | Headless mode |
| `/help` | Show help |
| `/clear` | Clear context |
| `/compact` | Compress context |

### Model Selection

| Shortcut | Model |
|----------|-------|
| `/model opus` | Opus 4.5 |
| `/model sonnet` | Sonnet 4.5 |
| `/model haiku` | Haiku 4.5 |
