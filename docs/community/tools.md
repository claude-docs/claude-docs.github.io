# Community Tools

Explore tools built by the Claude Code community.

## Multi-Agent Orchestration

### Claude Squad

Terminal app for managing multiple Claude instances:

```bash
npm install -g claude-squad
claude-squad init
```

[GitHub](https://github.com/smtg-ai/claude-squad)

### Claude-Flow

Advanced multi-agent orchestration with 54+ specialized agents:

```bash
npm install -g claude-flow
claude-flow init
```

[GitHub](https://github.com/ruvnet/claude-flow)

## Conversation Tools

### Claudex

Web-based conversation history browser:

- View past sessions
- Search conversation history
- Export conversations

[GitHub](https://github.com/user/claudex)

### Claude History

CLI tool for managing conversation history:

```bash
npm install -g claude-history
claude-history list
claude-history export
```
